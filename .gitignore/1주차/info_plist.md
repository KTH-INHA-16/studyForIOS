# info.plist

bundle을 식별하고 구성하는 key와 value로 쌍으로 이루어진  resource

app, framework, 또는 다른 entity를 key - value로 표현

대분류로 Core Foundation key, Launch Services Key, Cocoa Key, MacOS Key, IOS Key, watchOS Key, App Extension Key로 분류된다.
